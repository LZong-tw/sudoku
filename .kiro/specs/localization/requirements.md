# Localization Requirements (zh-TW)

## Overview
Defines requirements for Traditional Chinese (Taiwan) localization.

## Requirements

### REQ-L10N-001: Use zh-TW Terms
- **Description**: Use Taiwan Traditional Chinese terms
- **Term Mappings**:
  - 設定 (not 設置)
  - 統計資料 (not 統計數據)
  - 復原 (not 撤銷)
  - 重做 (not 恢復)

### REQ-L10N-002: Difficulty Labels
- **Description**: Difficulty names in zh-TW
- **Labels**:
  - easy: 簡單
  - medium: 中等
  - hard: 困難

### REQ-L10N-003: UI Labels
- **Description**: All UI text in zh-TW
- **Labels**:
  - 提示, 筆記, 新遊戲, 檢查
  - 成就, 每日挑戰
